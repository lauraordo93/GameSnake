# 🐍 Juego Snake en Java

Este es un proyecto del clásico juego **Snake**, desarrollado en Java utilizando la biblioteca **Swing**.  
El juego cuenta con niveles progresivos, trampas aleatorias y una velocidad que aumenta dinámicamente.

---

## 🎮 Cómo jugar

Usa las teclas de flechas para mover la serpiente:

- ⬆️ Arriba  
- ⬇️ Abajo  
- ⬅️ Izquierda  
- ➡️ Derecha  

Come la comida roja 🍎 para ganar puntos.  
Cada 6 puntos subes de nivel y aparecen nuevas trampas ⚠️.  
¡Evita chocar con tu propio cuerpo, los bordes o una trampa!

---

## ✨ Características

- 🎞️ Animación fluida con `javax.swing.Timer`  
- 🔁 Sistema de niveles automático cada 6 puntos  
- ⚔️ Trampas generadas aleatoriamente  
- 🚀 Dificultad progresiva (mayor velocidad)  
- ✅ Juego completamente funcional desde cero  

---

## 🧰 Tecnologías utilizadas

- Lenguaje: Java 17 o superior  
- Interfaz gráfica: Swing (`JPanel`, `JFrame`, `Timer`)  
- IDE recomendado: IntelliJ IDEA
